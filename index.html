Enterprise Prototype

Notification Management Prototype

An end-to-end approval workflow for high-stakes notifications, built from a written PRD to a clickable prototype.

Problem

Teams need to send urgent notifications (e.g. outage alerts) to different user groups, but high-stakes messages require stakeholder approval first — with a full audit trail of who approved what, and when.

Why AI

Used Claude to go from a written business requirements doc straight to a working, clickable prototype — confirming scope and page plan before building, to skip the design-to-dev handoff cycle and validate the workflow with stakeholders in hours, not weeks.

Stack
Claude HTML CSS JavaScript
Cost
$0 — static file, no hosting or API cost
Description

Compose a notification, route it to primary/secondary approvers, simulate the approval email with Accept/Reject actions, and log every notification's status — Pending, Approved, Rejected, Sent — in a searchable history table.

Newsletter Digest Agent

An automated pipeline that summarizes newsletters so nothing valuable gets lost in an unread inbox.

Problem

Stack

Cost
~$0.01–0.03 per newsletter summarized

Personal Finance Dashboard

A local-only dashboard built around one hard constraint: financial data never leaves the device.

Problem

Wanted a monthly view of net worth and account balances without handing financial data to any third-party app or cloud service.

Why AI

Used Claude to design and build around a non-negotiable privacy requirement — translating "data must never leave the device" into concrete guardrails: no cloud calls, no browser storage, local-only parsing.

Stack
HTML CSS JavaScript Chart.js SheetJS
Cost
$0 — fully static, in-browser only
Description

A single self-contained HTML dashboard that parses a monthly spreadsheet export entirely in the browser and renders account and net-worth charts — no login, no server, no data transmission.

Stock Research Agent

An on-demand research assistant that consolidates data and news into one readable report.

Problem

Researching a stock or ETF before investing means manually checking multiple sites and reading scattered news — a repetitive, time-consuming process.

Why AI

Used Claude to scope the full system, weigh data-source tradeoffs like free API limits versus paid reliability, and design the synthesis logic that turns raw data into a readable summary.

Stack
Python Telegram Bot API yfinance SEC EDGAR Tavily Claude API
Cost
Mostly free — ~$0.01–0.03 per report generated
Description

Triggered via a Telegram message with a ticker and time horizon, the agent pulls financial data and recent news, then uses Claude to synthesize a clean, brief-style summary delivered by email.
